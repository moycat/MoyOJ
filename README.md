判题实现
-----------

1. 网页端接收用户的提交，进行预处理后存入数据库
2. 判题端检测到数据库中有指派到当前评测端评测的提交，下载源码及测试数据
3. 建立一个限制容量的映像，写入源码和输入数据，启动Docker容器并挂载此镜像
4. 容器中的程序对选手源码进行编译/运行/控制，结果写入镜像后结束容器
5. 判题/通信程序进行判题，然后将结果存入数据库
6. 网页端可以看到评测结果
